# Personal Resume Site

A responsive, single-page resume website built with Next.js and Tailwind CSS. Designed to present my education, skills, and background in a clean and professional format.

## About the Project

This project is a personal resume page that displays my background as a Business Analytics & Information Systems student at USF. It features a structured layout with sections for education, skills, and contact information, all styled using utility-first CSS with Tailwind.

## Technologies Used

- [Next.js](https://nextjs.org/) – React framework for building the app and handling routing
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework for styling
- [TypeScript](https://www.typescriptlang.org/) – Type-safe JavaScript
- Semantic HTML5 – Used throughout for accessibility and structure

## Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/Bornzine/nextjs-resume.git

# Navigate into the project folder
cd nextjs-resume

# Install dependencies
npm install

# Start the development server
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Clean, single-page layout optimized for desktop viewing
- Sections for education, skills, and contact information
- Styled entirely with Tailwind CSS — no separate CSS files needed
- Semantic HTML structure using `<header>`, `<main>`, `<section>`, and `<footer>` tags
