# My Resume Site

This is my resume page built with Next.js and Tailwind CSS for ISM 3232. Nothing too crazy — just a clean single page with my info on it.

## Running it locally

Clone the repo, then run:

```bash
npm install
npm run dev
```

Should pop up at http://localhost:3000.

## What I built

- My name, contact info, and a short bio at the top
- Education and experience sections
- Skills listed out at the bottom
- Used semantic HTML tags throughout (header, main, section, footer)
- Styled everything with Tailwind — no separate CSS file

## A few things to note

I don't have a ton of work experience yet so that section is pretty bare, I'll fill it in more over time. The layout looks best on a desktop screen.
